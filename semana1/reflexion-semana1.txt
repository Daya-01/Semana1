El uso responsable de la información en internet es clave para participar como ciudadano digital de manera ética y segura. Cuando verificamos las fuentes, evitamos compartir desinformación y contribuimos a un debate más sano. La participación ciudadana se fortalece al usar datos confiables para opinar y tomar decisiones. Git y GitHub ayudan a organizar el trabajo técnico mediante el control de versiones y la colaboración estructurada. La gestión de cambios permite documentar lo que se hace y compartirlo con otros. Además, estas herramientas fomentan la transparencia y el orden en proyectos de grupo. Usar Git ayuda a que el trabajo sea reproducible y fácil de revisar. En conjunto, estas prácticas hacen más responsable y efectiva la acción en línea.
